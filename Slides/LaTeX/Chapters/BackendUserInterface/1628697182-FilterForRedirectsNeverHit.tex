% Feature | 94489 | Filter for redirects never hit

\begin{frame}[fragile]
	\frametitle{Backend User Interface}
	\framesubtitle{Weiterleitungen}
	\small
	Backend-Benutzer können jetzt eine Option "never hit" in \textbf{Site Management -> Redirects} aktivieren.
	Dieser Filter zeigt nur Weiterleitungen an, die nie aufgerufen wurden.\newline
	\small
		Hinweis: Die Funktion benötigt, dass die Option "redirects hit count" aktiviert ist.
	\normalsize
