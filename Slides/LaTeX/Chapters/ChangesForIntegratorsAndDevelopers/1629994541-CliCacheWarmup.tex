	\begin{itemize}
		\item TYPO3's command line interface (CLI) supports a new command:
\begin{lstlisting}
./bin/typo3 cache:warmup [--group <all|system|di|pages|...>]
\end{lstlisting}
		\item Specific cache groups can be defined via the group option.
	\end{itemize}
